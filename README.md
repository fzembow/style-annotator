# Style annotator
Automated grading utilities for CS50
Author: Fil Zembowicz     (fil@filosophy.org)

### Dependencies
to run the server:
Flask
    easy_install Flask
Mako templating
    easy_install mako

### Usage
Best way to check out how this works is to run the server, using

    python server.py

Then, visit the page and paste .c code and check the output.
